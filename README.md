## 纳新网站后台部分源码
----

### 前端接口
* /mail         邮件通知
* /getInfo      获取信息（登录状态下）
* /login        登录
* /verCode      验证码
* /check        检查（报名状态查询）
* /add          报名（登录状态下）
* /returnInf    教务系统信息


### 管理端接口
* /delete       删除
* /update       更新（管理端用）
* /all          管理（返回信息、需要管理员密码）

请求参数和返回参数根据需求文档<br>
使用session服务， GET/POST/JSONP
